# DevTeam2-TestServer

소리 시각화를 위한 테스트 서버입니다.

## 환경
- Node.js
- Express
- WebSocket (ws)

## 패키지 설치

```bash
npm install express ws
```
Node.js가 설치되어 있어야 합니다.

## 실행 방법

```bash
node test-server.js
```

## 주의 사항
1. 서버를 실행하는 컴퓨터와 Meta Quest 기기가 같은 로컬 네트워크에 연결되어 있어야 합니다.
2. 현재 서버가 실행 중인 컴퓨터의 로컬 IPv4 주소(예: 192.168.x.x)를 미리 확인해두세요.

## 참고 사항
- 본 서버는 테스트 및 데모 목적으로 사용됩니다.
- 실제 서비스 환경에서는 보안 및 성능 최적화가 필요합니다.
